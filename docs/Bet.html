Class: Bet

Bet(product, selections, stake)

new Bet(product, selections, stake)

A class to handle various types of bets. This class exposes a create method (a factory method) to create a bet based on its type while still being the actual Bet object for inherited classes.
Parameters:
Name Type Description
product string Product type of bet W, P or E
selections string | Array.<string> Selections of runners for the bet
stake string Stake for the bet
